Un mystère de Ladies Learning Code

Où sont les femmes dans le domaine de la technologie?

Vous arrivez à l'atelier.

Il y a quelques semaines, vous vous êtes inscrite à votre tout premier cours de programmation avec Ladies Learning Code. Ce matin, vous étiez si excitée de venir que vous êtes arrivé à l'avance!

Quand vous êtes là, vous sonnez à la porte. Tout ce que vous pouvez entendre derrière la porte, c'est du silence. Mauvais signe. Comment êtes-vous censée entrer?

C'est à ce moment que vous remarquez que la porte d'entrée est légèrement entrouverte. Vous entrez quand même: pas question de manquer l'atelier!
