Que se cache-t-il dans le placard?

Votre curiosité l'emporte et vous entrez dans le placard illuminé.

En approchant, vous voyez immédiatement la source de lumière: un téléviseur qui est resté allumé avec le générique de fin d'un film à l'écran.

Sur la table à côté du téléviseur, vous apercevez le boîtier du DVD. On dirait un genre de documentaire. Comme il a l'air intéressant, vous avez envie de le regarder...
